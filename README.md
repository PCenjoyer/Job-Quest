## Описание

**JobQuest** — это веб-приложение для управления процессом поиска работы. Приложение помогает соискателям:

-  **Отслеживать вакансии** — ведите подробный список компаний и должностей, на которые вы откликались
-  **Управлять статусами** — отслеживайте этап каждой заявки (отправлено, на рассмотрении, приглашение, отказ, предложение)
-  **Планировать интервью** — фиксируйте даты и время проведения собеседований
-  **Добавлять заметки** — храните информацию о контактах, требованиях к должности и впечатления от компании
-  **Анализировать прогресс** — визуализируйте статистику по откликам и успешным попыткам

---

##  Основные возможности

### Управление вакансиями
- Создавайте и редактируйте записи о вакансиях
- Присваивайте статусы (новое, рассматривается, интервью, предложение, отказ)
- Сортировка и фильтрация по различным критериям
- Импорт/экспорт данных

### Отслеживание интервью
- Ведите расписание собеседований
- Напоминания о предстоящих интервью
- История проведённых собеседований

### Личные заметки
- Добавляйте контактную информацию рекрутеров
- Сохраняйте требования к должности
- Записывайте ваши впечатления после интервью

### Аналитика и отчёты
- Статистика откликов по дням/неделям
- Процент успеха по компаниям
- Диаграммы и графики

## Быстрый старт

### Требования
- Node.js 16+
- npm или yarn

### Установка

1. **Клонируйте репозиторий**
```bash
git clone https://github.com/[ваш-username]/job-quest.git
cd job-quest
```

2. **Установите зависимости**
```bash
npm install
# или
yarn install
```

3. **Запустите приложение**
```bash
npm run dev
# или
yarn dev
```

4. **Откройте в браузере**
```
http://localhost:3000
```

## Документация

### API Endpoints

#### Вакансии
```
GET    /api/jobs           # Получить все вакансии
POST   /api/jobs           # Создать новую вакансию
GET    /api/jobs/:id       # Получить вакансию по ID
PUT    /api/jobs/:id       # Обновить вакансию
DELETE /api/jobs/:id       # Удалить вакансию
```

#### Интервью
```
GET    /api/interviews     # Получить все интервью
POST   /api/interviews     # Создать новое интервью
PUT    /api/interviews/:id # Обновить интервью
DELETE /api/interviews/:id # Удалить интервью
```

#### Статистика
```
GET    /api/stats          # Получить статистику
```
